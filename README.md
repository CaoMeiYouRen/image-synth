## 🏠 主页

[https://github.com/CaoMeiYouRen/image-synth#readme](https://github.com/CaoMeiYouRen/image-synth#readme)


## 📦 依赖要求


- node >=18

## 🚀 安装

```sh
npm install image-synth
```

## 👨‍💻 使用

```sh
npm run start
```

## 🛠️ 开发

```sh
npm run dev
```

## 🔧 编译

```sh
npm run build
```
